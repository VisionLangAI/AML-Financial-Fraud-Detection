# FinBERT AML Framework

This repository presents a complete transformer-based Anti-Money Laundering (AML) detection framework using the SAML-D financial transaction dataset. The proposed system utilizes FinBERT for contextual financial transaction representation learning and compares its performance against conventional machine learning, deep learning, and transformer-based baselines including DT, RF, SVM, XGBoost, LightGBM, LSTM, Bi-LSTM, TabTransformer, and FT-Transformer.

The framework includes:
- Data preprocessing and feature engineering
- Label encoding and feature standardization
- Feature selection using Information Gain and Gain Ratio
- Comprehensive model benchmarking
- Confusion matrix and ROC-AUC analysis
- Training and validation learning curves
- Ablation study analysis
- Statistical significance testing
- Explainable AI analysis using LIME, SHAP, Integrated Attention Rollout, and Transformer Attribution Visualization

This repository is designed for reproducible AML research and transformer-based financial transaction intelligence analysis.
